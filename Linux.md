# Linux

## 网络连接的三种方式

![](https://github.com/Jomocool/Operator-System/blob/main/Linux-img/1.png)

1. **桥接模式：**虚拟系统可以和外部系统通讯，但是容易造成IP冲突
2. **NAT模式：**网络地址转换模式，虚拟系统可以和外部系统通讯，不造成IP冲突。但是外部系统无法和虚拟系统通讯
3. **主机模式：**独立的系统，不和外部系统通讯
